Introduction to Classes

Prototypes & Classes

Classes are another way of writing prototypes and constructor functions. The following two blocks are two ways to write nearly identical things:

function Person(name, age, height) { this.name = name; this.age = age; this.height = height; } Person.prototype.sayHello = function() { return 'Hi my name is ' + this.name; } class Person { constructor(name, age, height) { this.name = name; this.age = age; this.height = height; } sayHello() { return 'Hi my name is ' + this.name; } }

Using Classes

You can create objects from classes using the new keyword

Extending Classes

extends keyword

The extends keyword allows us to create a new class using another as the base

constructor() method

The constructor() method runs when we use the new keyword to create an object from a class.

Be Careful with "this"

The "this" keyword will always be the object that is calling the function
